Privacy Policy — Rataspe Bot

Last updated: 2025-01-01

Rataspe Bot respects user privacy and follows Discord’s Developer Policies. This Privacy Policy explains what information is collected, how it is used, and how users may request the removal of data.

1. Information We Collect

The bot only collects the minimum information required to operate:

  • Server ID
  • User ID
  • Channel ID
  • Command interactions

The bot does NOT collect any personal information such as:

  • Real names
  • Full message content
  • Email addresses
  • IP addresses
  • Any data outside of Discord

2. How We Use Information

The collected data is only used to:

  • Execute bot commands
  • Respond to user interactions (menus, buttons, etc.)
  • Maintain temporary operational logs

The bot does not sell, share, store, or transmit data to third parties.

3. Data Deletion

Rataspe Bot does not store data permanently. Therefore, there is no long-term history to delete. If you would like to request removal of temporary logs or confirm the data handled, you can contact us.

4. Security

All processed data stays within Discord’s infrastructure. The bot does not maintain external databases or servers for user information.
