Privacy Policy — Rataspe Bot

Última actualización: 2025-01-01

Rataspe Bot respeta la privacidad de los usuarios y cumple con las políticas de desarrolladores de Discord. Esta Política de Privacidad explica qué información se recopila, cómo se usa y cómo los usuarios pueden solicitar su eliminación.

1. Información que recopilamos

El bot solo recopila la información mínima necesaria para funcionar:

  • ID del servidor
  • ID de usuario
  • ID de canal
  • Uso de comandos

El bot NO recopila información personal como:

  • Nombres reales
  • Mensajes completos
  • Correos electrónicos
  • Direcciones IP
  • Datos fuera de Discord

2. Uso de la información

La información recolectada se utiliza únicamente para:

  • Ejecutar comandos del bot
  • Responder a interacciones (menús, botones, etc.)
  • Registros temporales de funcionamiento

El bot no vende, comparte ni almacena datos con terceros.

3. Eliminación de datos

Rataspe Bot no almacena datos permanentemente, por lo que no existe historial que eliminar. Si deseas solicitar eliminación de datos temporales o verificar qué maneja el bot, puedes contactarnos.

4. Seguridad

Todos los datos manejados por el bot permanecen dentro de la infraestructura de Discord y no se almacenan en servidores externos.
